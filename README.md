# Demo Thunderbird Overlay Extension

This extension alters the behaviour of the _Help_ menu item in Thunderbirdâ€™s menu bar. It replaces its original menu items with whatever the user has selected in the menu's preferences.

## Install

Requires Thunderbird 68+

- Clone the repo
- In Thunderbird select _Tools_ > _Add-ons_
- Select _Debug Add-ons_ from dropdown
- Click _Load Temproray Add-on..._
- Select `manifest.json` from repo
